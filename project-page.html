Extra View Modes supplies a curated set of entity view modes used by our extra_* recipes. These view modes enable a variety of teaser and listing presentations out of the box (featured teasers, metro tiles, slides, video grid items).

Usage

Install the recipe and then manage display settings under *Structure → Display modes → View modes*. Combine these modes with exported displays to create consistent front-end listings and teasers.

Installation

composer require drupal/extra_view_modes
# drush recipe extra_view_modes
